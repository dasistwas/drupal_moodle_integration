Drupal Moodle Integration


INTRODUCTION
------------
    1. Drupal Moodle Integration modules allow you to integrate with Moodle and get details based on the user logged in.    
    2. Create  webservices in Moodle.   
    3. Navigate to moodle configuration in configuration section and specify url ('http: //url>'/ http's: //url>) and token.	
    4. If the user is registered in Moodle try login with same username and password into a Drupal site.
    5. Users will be created in the drupal and will redirect to course page displaying all courses assigned to users.
